# Contributing

Thank you for your interest in this project.

This repository accompanies a paper currently under journal review. The scope of contributions is intentionally narrow during this period.

## What I welcome

- **Bug reports** — If the notebook fails to run (import errors, version conflicts, data generation issues), please open an issue with your Python version and the full error traceback.
- **Reproducibility issues** — If results differ significantly from those documented in the README, open an issue with your environment details.
- **Typos and documentation fixes** — PRs welcome, no issue required.

## What I am not accepting right now

- Changes to the core monitoring methodology — the framework is tied to a paper under review.
- New features or additional drift detection methods.
- Alternative model architectures.

## How to report a bug

Open an issue and include:

1. Your Python version (`python --version`)
2. Package versions (`pip freeze`)
3. The cell that failed and the full error message
4. Operating system

## Code style

If you submit a PR, please follow the existing style: numpy docstrings, functions over classes for stateless utilities, no external dependencies beyond those in `requirements.txt`.
